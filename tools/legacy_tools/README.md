# TGX Legacy Tools

This directory contains older TGX conversion scripts kept for reference and compatibility.

- `image_converter.py`: older image-to-header converter.
- `obj_to_h.py`: older OBJ-to-`Mesh3D` converter.
- `texture_2_h.py`: older texture converter.

For new projects, prefer the newer tools from the parent `tools/` directory.
