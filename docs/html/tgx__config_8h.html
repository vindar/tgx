#define TGX_USE_FAST_INV_TRICK   TGX_CONFIG_USE_FAST_INV_TRICK

Enable the fast approximate inverse trick on architectures where it is beneficial.

This is selected automatically from the target board/architecture below, unless explicitly overridden before including TGX headers or from the compiler command line.
